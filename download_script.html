현재 모드팀에 속하신 분들 입니다.
- 제조일자, GPS, 원본, You, 곱등연가, Aron, 멸종위기의 AB형, 하티브님
 
앞으로 더 모집 할 수도 있습니다.
 
 
1. 무단 배포 금지 (자신의 블로그에 올리지 마세요.)
  - 카페, 카스, 개인전달 또한 금지입니다.
2. 사용하시다가 버그를 발견하셨다면 바로바로 알려주세요!
3. 다운이 가능한 경우에도 덧글에 다운로드 주소를 쓰는 것 금지
4. 카페 채팅으로 대화는 일절 받지 않습니다.
 
 
모드를 받았는데 어떻게 쓰는지 모른다구요?
 
모드를 패치하는 방법엔 두가지가 있습니다.
1. BlockLauncher (모드, 텍스쳐팩, 스킨, IP패치(마크 내부 지원에의해 사라질 기능), 게임중 모드적용, 스크립트)
2. PocketTool (모드, 텍스쳐팩, 스킨, IP패치)
 
이 두가지가 존재하지만, 예전엔 포켓툴 이용자가 많았지만,
지금은 포켓툴보다는 블럭런쳐를 더 많이 사용하게 되었습니다.
 
블럭런쳐 사용법을 알려드리겠습니다.
링크 클릭 : http://cafe.naver.com/minecraftpe/1352674 (퓨우님 작성)
 
하지만 블럭 런쳐는 아시다시피 유료입니다.
체험판이 존재하긴 하나 그것도 싫으시다면,
 
포켓툴을 이용하는 방법밖엔 없습니다.
 
포켓툴은 두가지의 버그가 존재합니다.
1. 젤리빈에서의 에러
2. 마크 업데이트시의 에러
 
위과 같은 버그가 발생한다면
1.그냥 블럭런쳐 무료버전을 사용하는것이 좋습니다.
(구입하면 더 좋구요)

이쯤되서 포켓툴 사용방법을 알려드리겠습니다.

5. 지형을 쉽고 빠르게 편집!
 
 
구역 설정 : 철 칼로 구역을 터치하시면 됩니다.
 
명령어 :
/set <blockID> : 지정한 구역을 입력한 blockID로 설정합니다.
/replace <blockID1> <blockID2> : 지정한 구역에서 입력한 blockID1을 입력한 blockID2로 변경합니다.
